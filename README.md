# Password Generator

A laconic password generator built with JavaScript.

## Features:

- Generate random passwords
- Customize password length
- Include/exclude:
  - Lowercase letters
  - Uppercase letters
  - Numbers
  - Symbols
- Copy password to clipboard

## Technologies:

- HTML
- CSS
- JavaScript

## How to use:

1. Set the desired password length
2. Select the options you want
3. Click "Generate"
4. Copy the result by clicking to the emoji
